# Cynic

Cynic is a Lightweight Rack-based framework. Cynic provides a framework with a few tools built on the ideas of rails.

## Installation

Add this line to your application's Gemfile:

    gem 'cynic'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cynic

## Usage

```bash
cynic new <name>
```
